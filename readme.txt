Bring WEEVE.tt rewards system to your Woo store!

== Description ==

The weeve loyalty plugin allows our weeve merchants to extend their loyalty program to their online shoppers. The plugin directly communicates with our weeve servers to issue and redeem points via the customer’s registered phone number.

In order to activate the plugin the merchant must obtain API keys from weeve as per their package agreement with weeve.

== Configuration  ==
1.	After installing the plugin, a new menu item will appear “ weeve” Select weeve 
1.	Go to setting 
1.	Set the points ratio (Points in % (100 / 10%) = 10 pts) this will determine how many points are issued to the total amount spent.
1.	API Key: these are the keys provide by to the merchant by weeve. 
1.	Vouchers : you can set vouchers that can be redeem for additional points the higher up a customer goes in the tier.
